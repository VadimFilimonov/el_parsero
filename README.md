# El Parsero

Simple template for parsing

## Initial setup

```sh
git clone git@github.com:VadimFilimonov/el_parsero.git
cp .env.example .env
cp .pages.example .pages
```

## Run environment

```sh
make up
```

## Setup

```sh
make install
```

## Build

```sh
make build
```

## Requirements

- Python 3.8
- Pipenv
